It redirect every 404 response to the search page with a default result on what is in the url. It allow the visitor to have something he was looking for, without forcing a redirect to a page he didn't ask for.

== Installation ==
1. Click on install
2. When it's activated, it's functionnal, no option to play with.
3. Just deactivate the plugin if you need to pause the redirect.
